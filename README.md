AdBrain is a simple command-line chatbot that behaves like a senior performance marketer.  
You can talk to it about advertising strategy (Meta, Google, TikTok, etc.) and it will:

- Ask smart clarifying questions  
- Suggest channels, angles, and hooks  
- Generate example ad copy  
- Give structured recommendations  

---

## Requirements

- Python 3.9 or newer  
- An OpenAI API key with billing enabled  
- `pip` for installing packages  

---

## Setup Instructions

1. **Unzip or open this project folder**

2. **Install dependencies**
